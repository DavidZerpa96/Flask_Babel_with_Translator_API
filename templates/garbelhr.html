{{ _('Resumen del Proyecto') }}

{{ _('El Dashboard de Análisis de Rotación de Empleados') }} {{ _('es una solución interactiva desarrollada en Power BI para Garbel, una empresa farmacéutica ficticia, diseñada para empoderar al departamento de Recursos Humanos. Utilizando datos obtenidos de Kaggle, el dashboard proporciona un análisis profundo de las métricas de rotación de empleados y un análisis integral del desempeño del departamento de RRHH.') }}

{{ _('El objetivo principal es identificar las causas raíz de la rotación, detectar tendencias y desarrollar estrategias accionables para reducir la rotación, mejorar la retención y fomentar la satisfacción de los empleados.') }}

{{ _('Objetivos Principales') }}

  1. {{ _('Diagnosticar Causas de Rotación:') }} {{ _('Analizar los datos de los empleados para identificar los factores principales que impulsan la rotación, como la satisfacción laboral, los niveles de ingreso y la carga laboral, además de señalar roles y departamentos de alto riesgo.') }}
  2. {{ _('Soporte a la Toma de Decisiones Estratégicas:') }} {{ _('Proporcionar insights accionables para ayudar a los equipos de RRHH a desarrollar estrategias basadas en datos para reducir la rotación y mejorar la satisfacción de los empleados.') }}
  3. {{ _('Análisis Integral de RRHH:') }} {{ _('Ofrecer una vista holística de métricas clave como demografía de empleados, niveles de satisfacción, distribución de ingresos y desempeño por departamento, permitiendo análisis detallados y comparaciones.') }}
  4. {{ _('Escalabilidad y Adaptabilidad:') }} {{ _('Demostrar un diseño de dashboard replicable y modular que puede adaptarse a las necesidades de otras organizaciones o industrias.') }}
