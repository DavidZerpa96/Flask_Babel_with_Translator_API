RRHH (Power BI)

Dashboard de rotación y drivers de retención para priorizar acciones de RRHH con datos.

Power BI DAX Modelado Python

{{ _('Dashboard Interactivo') }}

{{ _('Recomendamos visualizar el panel en dispositivos de sobremesa o tabletas para una mejor experiencia.') }}

Contexto

Detectar señales de rotación y entender qué variables se relacionan con satisfacción y permanencia.

Qué construí

Un dashboard con segmentaciones y KPIs de rotación/satisfacción para analizar por rol, departamento y variables clave.

Valor aportado
  • Priorizar acciones de retención con evidencia.
  • Identificar colectivos de riesgo y drivers potenciales.
  • Comunicar insights de RRHH de forma visual y accionable.

  • Dataset público (Kaggle) para simular un caso real de HR analytics.
  • Modelado + medidas DAX para segmentación y KPIs.
  • Objetivo: dashboard replicable y modular para otros contextos.
Nota

Empresa y datos ficticios/públicos. El foco está en modelado, DAX y storytelling analítico para decisiones de RRHH.
